# Vacancy Aggregator

Консольное Java-приложение для автоматического сбора, обработки, хранения, поиска, фильтрации, экспорта и анализа вакансий из нескольких источников.

Приложение позволяет добавлять источники вакансий, запускать ручное или автоматическое обновление, сохранять найденные вакансии в локальную базу данных, исключать дубли, искать и фильтровать вакансии, просматривать статистику, настраивать уведомления и экспортировать данные в CSV, JSON и HTML.

## Технологии

- Java 25
- Maven
- JDBC
- H2 Database
- jsoup
- JUnit Jupiter

## Функциональность

Приложение поддерживает:

- добавление и настройку источников вакансий;
- ручное и автоматическое обновление вакансий;
- парсинг вакансий из HTML-страниц;
- сохранение данных в локальную базу данных H2;
- исключение дублей по ссылке на вакансию;
- поиск, фильтрацию и сортировку вакансий;
- просмотр краткой и полной информации о вакансии;
- настройку уведомлений о новых вакансиях;
- просмотр истории изменений;
- аналитику по категориям, городам и зарплатам;
- экспорт данных в CSV, JSON и HTML;
- запуск юнит-тестов.

## Сборка проекта

```bash
mvn clean package
```

После сборки jar-файл будет находиться в папке `target/`:

```text
target/vacancy-aggregator-1.0.0.jar
```

## Запуск приложения

```bash
java -jar target/vacancy-aggregator-1.0.0.jar
```

После запуска в консоль будет выведен текст:

```text
Vacancy Aggregator. Введите help для списка команд.
>
```

Для просмотра списка команд:

```text
help
```

## Запуск тестов

```bash
mvn test
```

## Быстрый запуск с демонстрационными данными

В проекте есть локальные HTML-файлы для демонстрации работы без доступа к интернету:

```text
examples/hh-demo.html
examples/habr-demo.html
```

После запуска приложения можно выполнить команды:

```text
add-source hh-demo HEADHUNTER "examples/hh-demo.html" 1
add-source habr-demo HABR_CAREER "examples/habr-demo.html" 1
refresh
list
stats
search Java
filter city=Москва minSalary=100000 sort=salary limit=10
view 1
export csv exports/vacancies.csv
export json exports/vacancies.json
export html exports/vacancies.html
history
```

## База данных

Приложение использует локальную базу данных H2.

Файл базы данных создается автоматически после запуска приложения:

```text
data/vacancies.mv.db
```

## Команды консольного интерфейса

| Команда                                   | Описание                     |
|-------------------------------------------|------------------------------|
| `help`                                    | список команд                |
| `sources`                                 | показать источники           |
| `add-source <name> <type> <url> [pages]`  | добавить источник            |
| `enable-source <id>`                      | включить источник            |
| `disable-source <id>`                     | выключить источник           |
| `delete-source <id>`                      | удалить источник             |
| `refresh`                                 | обновить вакансии            |
| `auto-refresh <seconds>`                  | включить автообновление      |
| `stop-auto-refresh`                       | остановить автообновление    |
| `list [limit]`                            | показать вакансии            |
| `search <keyword> [limit]`                | поиск вакансий               |
| `filter key=value ...`                    | фильтрация и сортировка      |
| `view <id>`                               | полная информация о вакансии |
| `add-alert key=value ...`                 | добавить правило уведомления |
| `alerts`                                  | показать правила уведомлений |
| `export <csv\|json\|html> <file> [limit]` | экспорт вакансий             |
| `stats`                                   | статистика                   |
| `history [limit]`                         | история изменений            |
| `exit` / `quit`                           | выход                        |
